Данный репозиторий создан для решения двух геометрических задач для лабораторной работы №1 студентами НГТУ НЭТИ группы АБ-320:
a.	Прямоугольник – периметр, площадь, длина диагонали;
b.	Треугольник – периметр, площадь по формуле Герона, проверка на равнобедренность;

В ветке triangle содержится работа, выполненная обучающимся Заякином М.М., направленная на решение задачи с треугольником. В данной части кода была проделана следующая работа: было создано две функции, помимо основной: 
1. Одна checkTrgl направлена на проверку данных, введенных пользователем и определение тем самым действительно ли эти данные могут служить сторонами для треугольника, составлена данная функция на основе математической теоремы, которая гласит, что сторона не может достигать и превышать сумму двух других сторон. 
2. Вторая функция checkSds необходима для проверки треугольника на равнобедренность, исходя из определения равнобедренного треугольника, что две стороны у такой фигуры должны быть равны (ВАЖНО: в общем случае равносторонний треугольник является равнобедренным, поэтому даже в таком случае будет выведено сообщение о том, что треугольник равнобедренный, т.к. это не противоречит принципам математики).

В ветке rectangle содержится работа, выполненная обучающимся Лужнецким А.А., направленная на решение задачи с прямоугольником (ВАЖНО: в данной задаче не производится проверка фигуры, т.к. заданные пользователем стороны будут подразумевать некоторый четырёхугольник, который согласно нашему заданию будем иметь все углы равные 90 градусам, а если стороны будут равны, что означает – квадрат, то данную фигуру мы воспринимаем как частный случай прямоугольника). Согласно правилам математики были решены задачи по нахождению периметра (сумма двух сторон, умноженная на два), площади (произведение двух сторон) и нахождение длины каждой диагонали (по правилам математики диагонали в прямоугольнике равны между собой и находятся при помощи теоремы Пифагора).
